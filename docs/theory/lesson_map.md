# Theory Lesson Map

The built-in theory module covers these lesson topics:

- What is machine learning?
- Supervised vs unsupervised learning
- Classification vs regression
- Training, validation, and test sets
- Feature scaling and normalization
- Encoding categorical variables
- Missing data handling
- Bias-variance tradeoff
- Overfitting and underfitting
- Cross-validation
- Classification metrics
- Regression metrics
- Interpretability and feature importance
- Neural-network basics
- Hyperparameter tuning and model selection

Suggested teaching flow:

1. Start with `What Is Machine Learning?`
2. Move to `Classification vs Regression`
3. Review preprocessing lessons before importing CSV data
4. Use the Hyperparameter Lab immediately after `Bias-Variance Tradeoff`
5. Use Compare Runs after `Model Selection`
